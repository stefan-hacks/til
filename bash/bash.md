## **1\. What is Shell Scripting?**[](#)

**Shell scripting** is writing a list of commands for your computer to run automatically.

**Think of it like this:**

- Normal: You type one command, wait, type another command, wait...
- Scripting: You write all commands in a file, then run them all at once!
